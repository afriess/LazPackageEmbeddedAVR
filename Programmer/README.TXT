Programmer targets are added by simply placing a file with the targets name into the subfolders. The target file comes without extension and contains only the command line for the programmer. Placeholders are as follows:

Identifier // Description // Example
---------------------------------------------
$(LAZWIZDevice) // device name // atmeg32
$(LAZWIZProject) // project name // Project1
$(LAZWIZPort) // programmer port // COM2
